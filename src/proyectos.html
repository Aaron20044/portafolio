ProjectTracker

Una aplicación de gestión de proyectos diseñada para equipos de desarrollo, con características de seguimiento de tareas, reportes de progreso y asignación de responsabilidades. Los usuarios pueden ver el estado de los proyectos, colaborar en tiempo real y recibir notificaciones sobre los cambios importantes.

Java (Spring Boot), MySQL, Tailwind CSS, React

HealthSync

Una plataforma de monitoreo de salud personalizada que permite a los usuarios registrar y hacer seguimiento de indicadores de salud (como presión arterial y ritmo cardíaco), programar recordatorios de medicamentos y acceder a reportes mensuales. Incluye integración con dispositivos de IoT para una actualización automática de datos.

ASP.NET Core, Azure, Bootstrap, Angular

EduConnect

Un sistema de gestión de aprendizaje para instituciones educativas. Permite a los profesores organizar cursos, cargar recursos educativos y calificar tareas, mientras que los estudiantes pueden visualizar su progreso y colaborar en foros de discusión. EduConnect también ofrece informes de rendimiento detallados para padres y administradores escolares.

Java (Spring Boot), PostgreSQL, Tailwind CSS, Vue.js

ASP.NET MVC, SQL Server, Bootstrap, jQuery
